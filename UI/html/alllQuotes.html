    Event Loop

    On : 25/09/2019

    The Event loop is flow used in Node to handle the asynchronous code that we write. When a program is executed, the node runtime creates a thread and all code are executed in that thread. Nodejs is an event based platform where everything happens is reactions to an event and the execution of callbacks is done by the event loop
